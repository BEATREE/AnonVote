# AnonVote

基于Paillier的匿名投票系统。

该仓库为项目前端仓库，后端项目为 `AnonVote-Server` 与 `Paillier-Server`

[关于系统](https://anonvote.beatree.cn/#/about)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
